# ashmithmaddala.com

Personal portfolio — static site built with vanilla HTML, CSS, and JavaScript. No frameworks, no build tools.

[Live](https://ashmithmaddala.com/)

## What's in it

- Intro greeting animation (cycles through languages, slides up)
- Interactive terminal with 50+ commands, plus argument-based commands like `open github`, `search appsec`, and `cat project portfolio`
- Particle canvas background
- Scroll reveal animations
- Project filter on the work page
- Fully responsive

## Structure

```
index.html          Homepage
about.html          About + experience + skills
work.html           Projects with filter
writing.html        Articles
contact.html        Contact links
404.html            Not found page
article-1.html      Article: API security
article-2.html      Article: Technical career
style.css           All styles
script.js           Particles, scroll reveal, project filter
terminal.js         Interactive terminal
intro.js            Greeting intro animation
public/             Images, resume, favicon
```

## Run locally

Open `index.html` in a browser. That's it.
